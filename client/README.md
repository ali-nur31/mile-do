# Mile-Do

A premium, modern task management application built with React, Tailwind CSS, and TanStack Query. Designed for productivity, aesthetics, and ease of use on any device.



## Features

- **Real-time Synchronization**: Powered by `TanStack Query` for instant UI updates and background fetching.
- **Smart Inbox**: A dedicated inbox for unfiled thoughts and tasks.
- **Dark Mode**: Fully supported beautiful dark theme for night-time usage.
- **Context Menus**: Native-feeling right-click menus for quick actions on tasks and lists.
- **Smart UX**: 
  - One-tap completion with optimistic updates.
  - Custom modals and toast notifications.
  - Animated sidebar and page transitions.
  - Responsive design optimized for mobile and desktop.
- **Tech Stack**: React 18, Tailwind CSS, Zustand, Axios, Vite, Framer Motion.

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
